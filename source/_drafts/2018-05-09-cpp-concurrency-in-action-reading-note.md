# C++并发编程-读书笔记

## 你好，C++的并发世界!

### 什么是并发
计算机领域的并发指的是在单个系统里同时执行多个独立的任务，而非顺序的进行一些活动

### 并发的途径
- 多进程
- 多线程（C++原生支持）
